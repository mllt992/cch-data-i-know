# 用户数据分析功能 - 快速上手指南

## 🚀 立即开始

### 1. 启动服务

```bash
# 如果已经在运行,跳过此步
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

### 2. 访问用户分析页面

打开浏览器访问: http://localhost:8000/users

## 📖 功能说明

### 左侧树形结构

- **用户节点**: 显示用户名称、角色、启用状态
- **密钥节点**: 显示密钥名称、预览值、启用状态
- **渠道节点**: 显示可用的渠道/供应商

点击用户或密钥可展开/折叠子节点

### 右侧详情面板

**点击用户时显示**:
- 总调用数
- 成功率
- 总费用
- 独立模型数
- Token 使用量
- 平均费用/次

**点击密钥时显示**:
- 上述所有指标
- 模型使用统计图表
- 渠道使用统计图表
- 最近使用记录表格(50条)

### 时间范围筛选

- 默认显示近30天数据
- 可自定义开始日期和结束日期
- 点击"应用时间段"按钮更新数据

## 💡 使用技巧

1. **快速查看总览**: 直接点击用户名,查看该用户所有密钥的汇总数据

2. **深入分析**: 点击具体密钥,查看该密钥的模型分布、渠道使用情况

3. **对比分析**: 切换不同密钥查看数据差异

4. **时间分析**: 调整时间范围,观察不同时期的使用趋势

## 🔍 API 使用示例

### 获取用户树

```bash
curl http://localhost:8000/api/users/tree
```

### 获取用户统计(最近7天)

```bash
curl "http://localhost:8000/api/users/10/stats?start_date=2026-02-26&end_date=2026-03-05"
```

### 获取密钥详细统计

```bash
curl "http://localhost:8000/api/users/10/keys/16/stats?records_limit=100"
```

## 📊 数据说明

- **成功率**: `(成功请求数 / 总请求数) * 100%`
- **费用**: 以美元(USD)为单位
- **Token**: 输入Token + 输出Token 总和
- **状态码**: 200 = 成功, 其他 = 失败

## ⚙️ 自定义配置

如需调整查询性能或添加缓存,请修改 `app/stats_service.py`:

- `get_users_tree()`: 用户树查询逻辑
- `get_user_stats()`: 用户统计查询
- `get_user_key_stats()`: 密钥统计查询

## 📝 常见问题

**Q: 为什么有些密钥没有渠道?**
A: 密钥的 `provider_group` 字段为空或未匹配到启用的渠道

**Q: 数据不准确?**
A: 请检查:
1. 时间范围是否正确
2. 数据库中是否有对应时间段的记录
3. 尝试点击"刷新数据"按钮

**Q: 加载很慢?**
A: 大量用户时可能较慢,建议添加缓存或分页功能
